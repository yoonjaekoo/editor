# C++ Static Editor for GitHub Pages

정적 웹앱으로 만든 C++ 코드 에디터입니다.

## 배포 방법 (GitHub Pages)

1. 이 저장소를 GitHub에 push
2. `Settings` → `Pages`
3. `Build and deployment`에서 Source를 `Deploy from a branch`로 선택
4. 브랜치(예: `main`)와 폴더(`/root`)를 선택 후 Save
5. 잠시 후 발급된 URL로 접속

## 기능

- Ace Editor 기반 C++ 편집기
- 자동 저장(localStorage)
- 복사 / `.cpp` 다운로드 / 예제 복원
- 타이핑 애니메이션(Word 스타일 커서 효과)
